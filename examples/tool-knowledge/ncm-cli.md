# ncm-cli — example memory

- [2026-05] `ncm-cli` 搜索成功，不代表播放一定成功；很多歌会死在“暂无音源或暂无播放权限”。  [HIT:1]
- [2026-05] 遇到这类失败时，不要继续一首一首盲补；先承认问题是音源可用性，不是命令格式。  [HIT:1]
- [2026-05] 某些版本的 `search song` 参数顺序很挑，优先用 `--keyword "<query>" --output json` 这种显式写法。  [HIT:1]
- [2026-05] 在当前环境里，`ncm-cli` 搜索如果不走登录 shell 或代理环境，容易直接报网络异常。  [HIT:1]
- [2026-05] 读播放器状态前可以先看 `bg-worker.log`，很多“没声音”问题其实早就写成了明确错误。  [HIT:1]
